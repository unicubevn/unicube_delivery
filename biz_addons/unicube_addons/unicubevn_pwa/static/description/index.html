Customize built Odoo 17 PWA

This module help to customize the builtin Odoo 17 PWA like: app name, app shortname, background and theme color ... and push notification through firebase.

    Instructions:
  • For PWA Setting: After install the module, go to Setting/PWA to config the PWA settings:
    • For Push Notification (Firebase) Setting:
    • Step 1: Setup your firebase project and enable the Cloud Messing API and Firebase keys
    • Step 2: Go to Setting/PWA to config the PWA settings then add firebase keys and firebase project informations
    • Step 3: Use Discuss app to test the Push notification.
